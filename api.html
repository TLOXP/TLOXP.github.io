API Documentation

Normal TLO Lookup

Endpoint: http://api.ching.lol

API Parameters

Parameter Description Example
fname The first name of the person. John
lname The last name of the person. Doe
street The street address. 1234 Sesame St
city The city name. Los Angeles
state The state abbreviation. CA
zipcode The ZIP code. 99999
dob The date of birth in MM/DD/YYYY format. 01/30/1999
license_key The license key for accessing the API. key_here

Example API request:

http://api.ching.lol?fname=John&lname=Doe&street=1234+Sesime+St&city=San+Diego&state=CA&zipcode=20593&dob=10/10/1999&license_key=key_here

Lookup Phone Number

Endpoint: http://api.ching.lol/number.php

API Parameters

Parameter Description Example
number The phone number to lookup. 888-888-8888
license_key The license key for accessing the API. key_here

Example API request:

http://api.ching.lol/number.php?number=888-888-8888&license_key=key_here

Get User Balance

Endpoint: http://api.ching.lol/get_balance.php

API Parameters

Parameter Description Example
user The user identifier or license key. key_here

Example API request:

http://api.ching.lol/get_balance.php?user=key_here
