API Documentation

API endpoint: http://api.ching.lol

API Parameters

Parameter Description Example
fname The first name of the person. John
lname The last name of the person. Doe
street The street address. 1234 Sesime St
city The city name. San Diego
state The state abbreviation. CA
zipcode The ZIP code. 20593
dob The date of birth in MM/DD/YYYY format. 10/10/1999
license_key The license key for accessing the API. key_here

API Example

Example API request:

http://api.ching.lol?fname=John&lname=Doe&street=1234%20sesime%20st&city=san%20diego&state=CA&zipcode=20593&dob=10/10/1999&license_key=key_here
